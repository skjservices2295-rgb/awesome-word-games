## Word Unscramblers

- [WordUnscrambler.tips](https://wordunscrambler.tips) - Free word unscrambler with 14 tools, 104K-word dictionary, and Scrabble/WWF scores on every result. No ads, no signup.
- [WordFinder](https://wordfinder.yourdictionary.com) - Word finder with dictionary integration.
- [Unscrambled Words](https://www.unscrambledwords.com) - Simple unscrambler with advanced filters.

## Scrabble Tools

- [WordUnscrambler.tips Scrabble Helper](https://wordunscrambler.tips/scrabble-word-finder/) - Scrabble word finder showing point values per letter and per word.
- [Hasbro Scrabble Dictionary](https://scrabble.hasbro.com/en-us/tools#dictionary) - Official Scrabble dictionary checker.

## Wordle Tools

- [WordUnscrambler.tips Wordle Solver](https://wordunscrambler.tips/wordle-solver/) - Wordle solver with letter position filtering and frequency analysis.
- [NYT Wordle](https://www.nytimes.com/games/wordle/) - The original Wordle game by the New York Times.

## Crossword Tools

- [WordUnscrambler.tips Crossword Solver](https://wordunscrambler.tips/crossword-solver/) - Crossword helper with pattern matching (use ? for unknown letters).

## Anagram Solvers

- [WordUnscrambler.tips Anagram Solver](https://wordunscrambler.tips/anagram-solver/) - Find all anagrams with Scrabble and WWF scoring.
- [Internet Anagram Server](https://wordsmith.org/anagram/) - Classic anagram generator.

## Dictionary & Reference

- [WordUnscrambler.tips Dictionary](https://wordunscrambler.tips/dictionary/hello/) - 10,000+ word definitions with letter analysis, rhyming words, and game strategy tips.
- [Merriam-Webster](https://www.merriam-webster.com) - Standard English dictionary.
- [OneLook](https://www.onelook.com) - Multi-dictionary search engine.
